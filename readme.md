# Shoe_Shop_Management_System
## Introduction
- The system is built to manage online shoe buying and selling at shoe shop
- Frontend : HTML, CSS (Bootstrap), JS
- Backend : PHP

## Feature
- Customer : 
  - Login/Signin 
  - Order products
  - Manage personal information
  - View order status 
- Manager :
  - Login/Signin
  - Manage(add, remove, update) products
  - View order status
